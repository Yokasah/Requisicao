# MICROSOFT AZURE LOGIN BOOKING VEHICLES WEBSITE

1. This Website is made in Portuguese but you can understand mostly how it works.

2. Its a simple Website that uses Microsft Azure as a base of login which in the "login.php" part haves explaining what you need to change to work

3. It uses a Email System to notify the Admin about the booking from MailTrap that its also explained

4. The Admin can edit everything and add new Vehicles as well

5. The booking will only work from Vehicles that are inserted into the Database
